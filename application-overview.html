Application Overview (to be updated)


This application will be a discord bot. Discord bots are a really cool way of making use of Discord.

If the bot is being hosted online, anybody with access to the bot can use it! It is up to the developer to make the many commands available to the user.

With 'Teemo-Bot' I will be implementing a variety of commands that the user will be able to make use of when they have the bot join their discord server.

The user will be able to simply type commands into their discord server channel and as long as Teemo-Bot is present, these commands will be executed on.
